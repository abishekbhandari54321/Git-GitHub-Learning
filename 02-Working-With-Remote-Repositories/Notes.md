Today's learning:

1. git clone

Purpose: Download a complete repository from GitHub.

Real-life example: You join a company and need to download the company's project.

2. git fetch

Purpose: Check for updates from GitHub without changing your files.

Real-life example: You want to see if your teammate pushed new code before updating your own files.

3. git pull

Purpose: Download and apply updates from GitHub.

Real-life example: Your teammate added a new feature and you want it on your computer.

4. git push

Purpose: Upload your commits to GitHub.

Real-life example: You finished a feature and want your teammates to see your code.
